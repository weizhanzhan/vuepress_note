设备综合效率，Overall Equipment Effectiveness，简称OEE。
OEE = 时间可动率 * 性能可动率 * 合格品率

OEE=合格品数量*生产节拍/计划生产时间 (注：效率计算公式同OEE)

时间可动率 = 运行时间 / 计划生产时间

性能可动率 = 理论生产时间/运行时间 (注：理论生产时间=(合格品数量+报废品数量)*生产节拍)

合格品率 = 合格品数量/(合格品数量+报废品数量)
